# Login and Registration System

## Description
This is a simple Login and Registration System developed in C++. It allows users to register with a usrename and password, 
stores the data in a text file, and verifies user credentials during login.

## Features
- User Registration
- User Login
- Duplicate Username Check
- File Handling using 'users.txt'
- Simple Console Interface

## Technologies Used
- C++
- File Handling

## How to Run
1. Compile the code:
   '''
   g++ main.cpp -o login
   '''
2. Run the program:
   '''
   ./login
   '''
   (On Windows: 'login.exe')

## Project Files
- 'mian.cpp' - Source code
- 'users.txt' - Stores registered usernames and passwords
